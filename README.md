**這是什麼？** 一套完整的線上課程自動化系統 — LP、LINE Bot、金流、追單全部串好。

你不用寫 code，**你只需要跟 AI 說話，它會幫你做所有技術操作**。

---

## 🚀 Step 1：第一次使用

打開電腦的 Terminal（終端機），貼這一行按 Enter：

```
claude
```

然後跟 AI 說：

> 幫我部署 AI 造局術

剩下的全部交給 AI。它會一步一步問你問題、一步一步做。你只要**回答問題**、**貼值給它**就好。

---

## 📝 Step 2：AI 會問你這些問題（先準備好）

部署過程中，AI 會**一次問一個**這些值。不用事先全找好，它問到才去查：

**基本資訊：**
- 你的品牌名（例：知脊整聊學院）
- 你的名字（會顯示在課程裡）
- 課程名稱（例：AI 造局術）
- 課程售價（數字，例：49800）

**LINE Bot：**
- LINE Channel Access Token
- LINE Channel Secret
- 你自己的 LINE User ID

**金流（TapPay）— 可以之後再設：**
- TapPay Partner Key / Merchant ID / App ID / App Key

**廣告（Meta）— 可以之後再設：**
- Meta Pixel ID
